Python CLI To-Do List

A simple command-line To-Do list application built with Python.

The program allows users to create, remove, and manage tasks directly from the terminal. Tasks are stored in a JSON file for persistence.

Features

Add tasks

Remove tasks

Clear the list

Save tasks to a JSON file

Simple CLI interface

How to Run

Clone the repository:

git clone https://github.com/yourusername/todo-list-python.git

Navigate to the project folder:

cd todo-list-python

Run the program:

python main.py

Project Structure

main.py – program entry point
funcs.py – CLI logic and task operations
storage.py – JSON storage management
